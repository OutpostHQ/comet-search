# @searchjs/css

Common CSS for Search.js components.

## Installation

```sh
npm install @searchjs/css
```

## Usage

With a bundler:

```sh
import '@searchjs/css';
```

With a CDN:

```sh
<link
  rel="stylesheet"
  href="https://unpkg.com/@searchjs/css@0.0.1/index.css"
/>
```
